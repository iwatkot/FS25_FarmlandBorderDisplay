## Overview

Farmland Border Display shows farmland borders directly in the world using marker points.

It helps with:
- planning land purchases,
- placing buildings/fences in build mode,
- checking owned vs unowned land at a glance.

Marker colors:
- Green: owned by you
- Cyan: unowned (buyable)
- Red: owned by another player

## Settings

Open `ESC > Settings` and configure:
- Show in build menu
- Show in game
- Show only owned
- Point size (%)
- Height offset (%)

## Installation

1. Download the mod ZIP.
2. Put the ZIP in:
  `Documents/My Games/FarmingSimulator2025/mods`
3. Start Farming Simulator 25 and enable the mod for your savegame.
